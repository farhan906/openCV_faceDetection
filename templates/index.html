Welcome to the Face Recognition System

Use the navigation menu to add a person or start face recognition.

The project aims to build a real-time face recognition system leveraging computer vision techniques. The core of the system is based on OpenCV for video capture and image processing, and the face_recognition library for facial feature extraction and matching. The system is designed to identify individuals from a pre-defined dataset and mark attendance automatically, making it suitable for use in educational institutions, workplaces, and security monitering.
